# **App Name**: Content Curator AI

## Core Features:

- Summarization: AI-powered tool to summarize content to essential information
- Keyword Extraction: Extract relevant keywords from web content using AI.
- Content Saving: Save summarized content and keywords locally, with import and export capability
- User Interface: Provide a simple UI to submit URLs, display summaries, and display keywords.
- Custom Prompts: Provide option for custom AI prompt to modify results, as needed.

## Style Guidelines:

- Primary color: Calm Blue (#64B5F6) to reflect clarity and focus in information processing.
- Background color: Light Blue (#E3F2FD) to maintain a clean and readable interface.
- Accent color: Warm Orange (#FFB74D) to highlight interactive elements and AI suggestions.
- Body and headline font: 'Inter', sans-serif, for clear and accessible readability of summaries and extracted keywords.
- Code font: 'Source Code Pro' for custom AI prompt display, if needed.
- Use clear, minimalist icons to represent various functions and content types.
- Use subtle animations for loading states and transitions to provide feedback without distraction.
